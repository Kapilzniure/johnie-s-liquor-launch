# Johnnies Liquor - Austin, Texas

A professional, mobile-first website for Johnnies Liquor, Austin's trusted liquor store offering wine, beer, and spirits at great prices.

## Business Details
- **Location**: 13201 Pond Springs Rd, Suite 203, Austin, TX 78729
- **Phone**: (512) 383-5004
- **Hours**: Mon–Sat, 10:00 AM – 9:00 PM
- **Instagram**: [@johnniesliquorstore](https://www.instagram.com/johnniesliquorstore/)

## Features
- Dark theme with gold accents
- Mobile-first responsive design
- Sticky bottom navigation for mobile
- Google Maps integration
- Clickable phone numbers
- Professional product categories
- Customer reviews section
- SEO optimized with structured data

## Tech Stack
- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui components
- Lucide React icons

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment
This website is ready for deployment to any static hosting service like Vercel, Netlify, or GitHub Pages.
